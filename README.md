#Project Overview

This project presents an HR Analytics Dashboard built using Microsoft Power BI to analyze employee data and identify key workforce trends. The dashboard helps HR teams understand employee attrition, workforce demographics, job satisfaction levels, and departmental performance.

The goal of this project is to support data-driven HR decision-making by visualizing important employee metrics and identifying factors that influence employee turnover
